## 📊 2. 사용한 데이터

### 1) 당뇨병 데이터 (Pima Indians Diabetes)
- **파일명**: `diabetes.csv`
- **데이터 수**: 768명
- **주요 컬럼**: 혈당(Glucose), 혈압(BloodPressure), BMI, 나이(Age) 등 8개 지표
- **결과 컬럼**: `Outcome` (0 = 정상, 1 = 당뇨병)

### 2) 알츠하이머 데이터 (Alzheimer's Disease Data)
- **파일명**: `alzheimers_disease_data.csv`
- **데이터 수**: 2,149명
- **주요 컬럼**: 인지검사(MMSE), 기능평가(FunctionalAssessment), 
  일상생활능력(ADL), 콜레스테롤, 혈압 등 30여 개 지표
- **결과 컬럼**: `Diagnosis` (0 = 정상, 1 = 알츠하이머)

## 📝 3. 프로젝트 안내

### 🎯 프로젝트 목표
질병에 걸린 환자와 정상인의 데이터를 비교하여,
**"어떤 특징이 질병과 깊은 관련이 있는지"**를 직관적으로 찾아냅니다.

### 🔍 주요 기능
1. **데이터 미리보기** – 전체 인원과 환자/정상인 비율을 확인
2. **평균 비교** – 두 그룹의 평균을 비교하고, 차이가 큰 순서로 정렬
3. **항목별 분포 비교** – 선택한 항목의 분포를 그래프로 시각화
4. **상관관계 히트맵** – 질병과 관련 깊은 항목만 골라서 표시

### 🛠️ 사용 기술
- **Streamlit** : 웹앱 제작
- **Pandas** : 데이터 처리 및 전처리
- **Matplotlib / Seaborn** : 데이터 시각화
- **NumPy** : 수치 계산

### 💡 데이터 전처리
- 의학적으로 불가능한 **0 값(혈당, 혈압 등)을 결측치로 처리** 후 중앙값으로 보완
- 분석에 의미 없는 컬럼(환자ID, 담당의사)은 제거
